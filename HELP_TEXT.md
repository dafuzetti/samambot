# Bot Help (English)

## New Event (creates it if needed)
- /join - Join the current event.
- /team-a @p1 @p2 ... - Add up to 4 players to team A.
- /team-b @p1 @p2 ... - Add up to 4 players to team B.
- /players @p1 @p2 ... - Add up to 8 players to the event.

## Results [gameloss] = 1 to report a 2x1
- /lose @winner [gameloss] - Report a loss and award a win to another player.
- /win @loser [gameloss] - Report a win and record a loss for another player.
- /result @winner @loser [gameloss] - Report a match result for other players.

## Other
- /movehere event_id - Move an existing event into this channel.
- /history event_id - List past events or show details for a specific event.
- /score @player - Show server scoreboard or a player's stats.

# Ajuda do Bot (Português)

## Novo Evento (cria o evento se necessário)
- /join - Entrar no evento atual.
- /team-a @p1 @p2 ... - Adicionar até 4 jogadores ao time A.
- /team-b @p1 @p2 ... - Adicionar até 4 jogadores ao time B.
- /players @p1 @p2 ... - Adicionar até 8 jogadores ao evento.

## Resultados [gameloss] = 1 para reportar um 2x1
- /lose @winner [gameloss] - Reportar uma derrota e registrar a vitória para outro jogador.
- /win @loser [gameloss] - Reportar uma vitória e registrar a derrota para outro jogador.
- /result @winner @loser [gameloss] - Reportar o resultado de uma partida para outros jogadores.

## Outros
- /movehere event_id - Mover um evento existente para este canal.
- /history event_id - Listar eventos anteriores ou mostrar detalhes de um evento específico.
- /score @player - Mostrar o ranking do servidor ou as estatísticas de um jogador.
